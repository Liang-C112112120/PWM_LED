# PWM_LED
使用兩個計數器交替運作來模擬PWM的訊號，控制LED燈的亮滅。

運用上次作業的兩個計數器來模擬PWM訊號，採用8位元的寬度，一個正數、一個倒數，兩者的計數值相加為255(2^8 = 255)。
